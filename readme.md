# 📘 Funções em JavaScript

## 📌 Descrição
Exercícios práticos para fixar conceitos de funções em JavaScript, com foco em lógica de programação e reutilização de código.

## 🚀 Conceitos
- Funções com e sem return
- Parâmetros e argumentos  
- Estruturas condicionais (if/else)
- Laços de repetição (for)
- Escopo local e global

## 💻 Funcionalidades
- Operações matemáticas (subtração, multiplicação, divisão)
- Verificação de número (par/ímpar, maior valor)
- Saudação personalizada
- Cálculo de idade
- Sistema de aprovação de aluno
- Tabuada de 1 a 10

## 📂 Estrutura
- funcao.js
- anotacoes.txt

## 🎯 Objetivo
Praticar lógica de programação e organização de código utilizando funções.
